Closures

Array

join(" ") gives a string of all elements of array push() to add last element of array and Pop() removes last element of array shift() removes first element of array and unshift() adds element to first position indexOf(x) starts looking for index from front and lastIndexOf(x) looks from end of array. Both use optional second parameter to specify position to start slice(x,y) gives an array starting from position x to position y. Important: x is inclusive and y is exclusive slice(x) gives array of all elements from x to end of array .concat() can be used to glue arrays together similar to what + does for strings Arguments object... All arguments passed to a function are stored in an arguments object which is similar to an array but does not have access to all the methodds that an array has function argumentsCounter() { console.log("You passed in ", arguments.length, " arguments"; }

The in operator can be used to check if the object contains a give property with a specific name It can also be used to loop over all properties of an object example (for( var name in object))

Strings

Can use .length .charAt(x) .trim() .toUpperCase .toLowerCase Also uses .slice(x,y) and string[x] similar to arrays .indexOf("a") or .indexOf("abc") can also work. Unlike arrays where indexOf can only look at 1 element

List of commands I used on git client

git clone from the "Clone or Download" button

cd JasonsPlayground To change directory to the local repo

git help Get to help file or documentation

git add -A It adds current content in the filepath to the index (so that we can commit new files/folders created on local) git commit -m This commited the files, but did not add new files (which is when I realized I need to do a git add) git push origin master For pushing files to the repo. But it gave an error, because I was on a different branch So modified to: git push origin git reset --hard Discard changes to local after last commit. Reset index and working tree. git pull origin To get latest from repo to local

Array

join(" ") gives a string of all elements of array push() to add last element of array and Pop() removes last element of array shift() removes first element of array and unshift() adds element to first position indexOf(x) starts looking for index from front and lastIndexOf(x) looks from end of array. Both use optional second parameter to specify position to start slice(x,y) gives an array starting from position x to position y. Important: x is inclusive and y is exclusive slice(x) gives array of all elements from x to end of array .concat() can be used to glue arrays together similar to what + does for strings

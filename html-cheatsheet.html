• h1 - h6 is for headings
  • p for paragraphs
  • br for line break
  • a href for links, target for window, alt for alt text
  • ul = unordered list, ol = ordered list, li = list item
  • use style for style attributes
  • use a period "." for class and hash # for id
  • input can be type="text" and have a placeholder
  • form can do an action like <form onClick()>
  • button could do action also like form
  • <link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css"/>
